
## Nueva Especificación de Página: "Revisión de IA"

La página tendrá una lista de tarjetas – una por cada email que requiere revisión. El diseño deja de usar columnas rígidas y pasa a un formato de **tarjeta vertical “plegable” (tipo split fold)** para mejorar claridad, navegación y experiencia de usuario.

### Estructura y Comportamiento

1. **Encabezado compacto y claro**
   - Cada tarjeta muestra en su parte superior la información principal del email

2. **Expandir detalles**
   - A la derecha del encabezado, habrá un **icono de flecha (chevron)** que permite al usuario expandir o contraer la tarjeta para ver detalles del análisis IA.
   - El área de expandir es un botón claro, separado de enlaces como “Ver detalle del email” para evitar errores de click.

3. **Análisis IA únicamente al expandir**
   - Cuando la tarjeta está expandida, se muestran los siguientes datos procesados por la IA
   - Si otra tarjeta estaba expandida, se pliega automáticamente para que sólo una esté abierta a la vez (tipo “accordion”).
   - Se agregan detalles visuales como fondo diferenciado y una transición suave para que el cambio sea legible, ordenado y no invasivo.

4. **Acciones simplificadas**
   - En la sección desplegada están los botones para aceptar o rechazar el análisis de la IA, bien separados y contrastados.

5. **Accesibilidad y usabilidad**
   - Los elementos interactivos tienen un área de click adecuada tanto para escritorio como para móvil.
   - El diseño prioriza la legibilidad y la facilidad de escaneo visual usando espacios en blanco y separación clara entre bloques de información.

6. **Vínculo al detalle completo**
   - El enlace para ver el detalle completo del email se mantiene accesible y separado del trigger para desplegar el análisis, evitando ambigüedad para el usuario.
